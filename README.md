# HelpDesk System - React Project

מערכת לניהול פניות (Tickets) המאפשרת למשתמשים לפתוח קריאות שירות, ולצוות התמיכה והניהול לטפל בהן ביעילות. המערכת מבוססת על React ומשתמשת ב-Context API לניהול מצב גלובלי.

## 📋 תיאור המערכת

האפליקציה מספקת ממשק משתמש מותאם אישית לכל סוג משתמש (Role), ומאפשרת זרימת עבודה מלאה של טיפול בטיקטים: החל מפתיחת הטיקט, דרך הקצאתו לסוכן, ועד לשינוי הסטטוס וסגירתו.

### תפקידים והרשאות

המערכת תומכת ב-3 סוגי משתמשים:

1.  **Customer (לקוח):**
    *   יכול לפתוח טיקטים חדשים.
    *   צופה רק בטיקטים שהוא יצר.
    *   יכול להוסיף תגובות לטיקטים שלו.

2.  **Agent (סוכן תמיכה):**
    *   צופה בטיקטים שהוקצו אליו לטיפול.
    *   יכול לעדכן סטטוס של טיקט.
    *   יכול להוסיף תגובות ולתקשר עם הלקוח.

3.  **Admin (מנהל):**
    *   צופה בכל הטיקטים במערכת.
    *   יכול להקצות טיקטים לסוכנים (Agents).
    *   יכול לנהל סטטוסים ורמות דחיפות.
    *   יכול לנהל משתמשים.

## 🛠️ טכנולוגיות

*   **Frontend:** React, TypeScript, Vite
*   **State Management:** Context API + useReducer
*   **Routing:** React Router Dom
*   **UI Library:** Material UI (MUI)
*   **HTTP Client:** Axios / Fetch
*   **Alerts:** SweetAlert2

## 🚀 הוראות התקנה והרצה

### דרישות קדם
*   Node.js מותקן.
*   שרת ה-API רץ ברקע (פורט 4000). יש להוריד ולהריץ את השרת מהקישור: [https://github.com/sarataber/helpdesk-api](https://github.com/sarataber/helpdesk-api)

### שלבי התקנה

1.  שכפל את המאגר (Clone):
    ```bash
    git clone <YOUR_REPO_URL>
    cd <YOUR_PROJECT_FOLDER>
    ```

2.  התקן את התלויות:
    ```bash
    npm install
    ```

3.  הרץ את הפרויקט:
    ```bash
    npm run dev
    ```

4.  פתח את הדפדפן בכתובת שתופיע בטרמינל (בדרך כלל `http://localhost:5173`).

## 📂 מבנה הפרויקט

*   `/src/components` - רכיבי UI (טפסים, רשימות, כרטיסים).
*   `/src/pages` - דפי האפליקציה המלאים.
*   `/src/context` - ניהול מצב גלובלי (AuthContext).
*   `/src/services` - קריאות לשרת (API).
*   `/src/assets` - תמונות וקבצים סטטיים.

## 🔐 פרטי התחברות (דוגמה)

ניתן להשתמש במשתמשים הקיימים בשרת או ליצור חדשים דרך ה-Swagger/Postman.
*   **Admin:** משתמש עם הרשאות ניהול.
*   **Agent:** משתמש עם הרשאות סוכן.
*   **Customer:** משתמש רגיל.
